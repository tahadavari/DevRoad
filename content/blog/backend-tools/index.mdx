علاوه بر زبان برنامه‌نویسی، ابزارهای مهم بک‌اند شامل این دسته‌ها هستند:

- IDE/Text Editor: مثل VSCode و Zed
- Database Tools: مثل DataGrip و Navicat
- API Tools: مثل Postman و Swagger
- Hosting/Deployment ابزارها
- Collaboration/Productivity: مثل Jira، Trello، iTerm و Raycast

در هر دسته گزینه‌های زیادی وجود دارد؛ بهترین راه، تست چند ابزار و انتخاب Stack مناسب تیم است.
