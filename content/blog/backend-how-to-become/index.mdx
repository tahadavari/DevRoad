اگر تازه‌کار هستی، از یادگیری یک زبان بک‌اند شروع کن: Python، Java، Go، Ruby و ... . بعد از مبانی زبان:

1. Package Manager همان زبان را یاد بگیر.
2. SQL و یک دیتابیس رابطه‌ای مثل PostgreSQL را یاد بگیر.
3. CRUD واقعی پیاده‌سازی کن.
4. یک Web Framework مناسب انتخاب کن.
5. یک RESTful API بساز.
6. Authentication/Authorization ساده اضافه کن.
7. Git و GitHub را جدی یاد بگیر.

بعد از این مرحله می‌توانی برای موقعیت‌های Junior اقدام کنی. داشتن پورتفولیو و پروژه واقعی اثر بسیار بیشتری از صرفاً دیدن دوره دارد.
